    Introduction

    Hi Folks,

    Today we are going to see how we can upload files to S3 with Golang. For this the prerequisites are,

    1. AWS Free tier account.
    2. IAM ID and Secret with S3 access.

    S3 is a cloud storage solution provided by AWS to store files. More info here.

    S3 Setup

    We will use the static credentials of an IAM user. We will use the env variables method, wherein we will set the ID and Secret and the AWS SDK will pick it from there when running the code.

    $ export AWS_ACCESS_KEY_ID=YOUR_AKID
    $ export AWS_SECRET_ACCESS_KEY=YOUR_SECRET_KEY

    More info is available here.

    We will now create a bucket named – s3-testing-2k22 (Note – S3 buckets are public so a unique name is required globally)

    Our Code

    Now let’s move to the code part,

    We need to create an AWS session. The session is a medium for connection. Many other AWS services can be used with the same session.

    We need to mention Region when creating the session, our ID and Secret will be picked from Environment as stated earlier.

    package main

import (
	"context"
	"log"
	"os"
	"time"

	"github.com/aws/aws-sdk-go/aws"
	"github.com/aws/aws-sdk-go/aws/session"
	"github.com/aws/aws-sdk-go/service/s3/s3manager"
)

func main() {

	ctx := context.Background()

	f, err := os.Open("test.txt")
	if err != nil {
		panic(err)
	}

	s3session := session.Must(session.NewSession(
		&aws.Config{
			Region: aws.String("ap-south-1"),
		},
	))

	uploader := s3manager.NewUploader(s3session)

	bucketName := "s3-testing-2k22"
	key := "folder1/" + time.Now().String() + ".txt"

	result, err := uploader.UploadWithContext(ctx, &s3manager.UploadInput{
		Bucket: aws.String(bucketName),
		Key:    aws.String(key),
		Body:   f,
	})

	if err != nil {
		panic(err)
	}

	log.Println("File Uploaded Successfully, URL : ", result.Location)

}

    Output

    Let’s run our code and see the output, We should get an URL for the file created.

    Lets look at the bucket from the AWS console,

    As we can see folder1 is created automatically and our file is also uploaded.

    S3 does not follow a folder hierarchy but it follows a key-value pair format. The key is the path to the parent folder and the value is the filename.

    Let’s update the Key as follows and see how the file is uploaded.

    key := “folder2/” + “folder3/” + time.Now().String() + “.txt”

    As we can see the nested folder is also created and the file is also uploaded.

    Conclusion

    To Conclude,

    We basically say how we can upload the file to S3 using Golang, and how to pass credentials to AWS, how to create a Session and folder nesting.

    Introduction

    Hey folks, Today we will see what Dynamo DB is, how to create an item in dynamo DB and how to retrieve it using Golang.

    It’s assumed that you already have an AWS account and have access to the Dynamo DB service. We are setting the ID and Secret in the environment. More info about that is in this post.

    DynamoDB is basically a No-SQL database, kind of like mongo. AWS does the infra management and we can directly use the service. More information is here.

    Similar to MongoDB collections, We have tables here. We need to set a partition key here, this key will be used to retrieve our value. Ideally, if we have an employee table, then the partition key would be the employee id.

    So below are the thing we will do today,

    1. Create a table in DynamoDB from Console & Code.
    2. Create an item from Console & Code.
    3. List our items from Console & Code.
    4. Delete our items from Code.
    This is a basic tutorial so we will just see the basic operations without expressions, conditions, etc.

    Create Dynamo DB Table

    We are going to use the console to create the table. Once we go to Dynamo DB in our management console, we can create a table. Our table creation page will look like below.

    Let us also see how we can create a table from our code.

    var awsSession = session.Must(session.NewSession(
	&aws.Config{
		Region: aws.String("ap-south-1"),
	},
)) //AWS Session object

var ddb = dynamodb.New(awsSession) //DynamoDB Session

func createTable(tableName string) {

	input := &dynamodb.CreateTableInput{
		AttributeDefinitions: []*dynamodb.AttributeDefinition{
			{ //We mention the key and its data type here.
				AttributeName: aws.String("empId"),
				AttributeType: aws.String("N"),
			},
		},
		//Mention if our attribute is partition key or sort key.
		//We use hash for partition key.
		KeySchema: []*dynamodb.KeySchemaElement{{
			AttributeName: aws.String("empId"),
			KeyType:       aws.String("HASH"),
		}},
		TableName:   aws.String(tableName),
		BillingMode: aws.String("PAY_PER_REQUEST"),
	}

	//create table is async, aws will give response early but may take some time to create the table.
	output, err := ddb.CreateTable(input)
	if err != nil {
		log.Println("error while creating table", err)
		return
	}

	log.Printf("Create table successful with name : %s , output is : %s", tableName, output)
}

    Create Dynamo DB item via Console & Code

    We can use the AWS Console to create items for our tables. It will look like the below image,

    Let us see how we can create an item from our Golang code. We create a putItemInput struct and call the putItem method with our struct. Our data types are not directly translated to DynamoDB data type, hence we need to use the AttributeValue struct.

    func createItem(tableName, empId, name, age string) {

	input := &dynamodb.PutItemInput{
		Item: map[string]*dynamodb.AttributeValue{
			"empId": {
				N: &empId, //N is number.
			},
			"name": {
				S: &name, //S is string
			},
			"age": {
				N: &age,
			},
			"active": {
				BOOL: aws.Bool(true),
			},
		},
		TableName: aws.String(tableName),
	}

	_, err := ddb.PutItem(input)
	if err != nil {
		log.Println("error while putting item", err)
		return
	}

	log.Printf("Put item success for empId : %s", empId)

}

    Get Dynamo DB item via Console & Code

    In the AWS console, listing the items is straightforward. We click on Explore Table Items button and we can see the data.

    Coming to our Go code, We are getting the item by passing the partition key empId. We get the item in the Item key of output. It will be nil if it does not exist.

    func getItem(tableName, empId string) {

	input := &dynamodb.GetItemInput{
		ConsistentRead: aws.Bool(true),
		Key: map[string]*dynamodb.AttributeValue{
			"empId": {
				N: &empId,
			},
		},
		TableName: aws.String(tableName),
	}

	output, err := ddb.GetItem(input)
	if err != nil {
		log.Println("error while getting item", err)
		return
	}

	//We get the data in the Item field. Item field is a map. In case the item does not exist the map is nil.
	if output.Item == nil {
		log.Printf("Item does not exist for empId : %s", empId)
		return
	}

	name := *output.Item["name"].S
	age := *output.Item["age"].N
	active := *output.Item["active"].BOOL

	log.Printf("Get item success, data :- empId : %s, name : %s, age : %s, active : %v", empId, name, age, active)
}

    Deleting Dynamo DB item via Code

    Deleting item from the AWS console is straightforward. We will see how we can delete items from the code. We get a DeleteItem method to delete items. We just need to pass the partition id here.

    func deleteItem(tableName, empId string) {

	input := &dynamodb.DeleteItemInput{
		Key: map[string]*dynamodb.AttributeValue{
			"empId": {
				N: &empId,
			},
		},
		TableName: aws.String(tableName),
	}

	_, err := ddb.DeleteItem(input)
	if err != nil {
		log.Println("error while deleting item", err)
		return
	}

	log.Printf("Delete item success for empId : %s", empId)
}

    Output

    Let’s see our main function. We will do some add, get, update, and delete operations here and see the output.

    func main() {

	log.SetOutput(os.Stdout)

	listTables()

	employeeTableName := "employee"

	// deleteTable(employeeTableName)

	// createTable(employeeTableName)

	//inserts
	createItem(employeeTableName, "1", "John Doe", "25")
	createItem(employeeTableName, "2", "Jane Doe", "20")

	//selects
	getItem(employeeTableName, "1")
	getItem(employeeTableName, "2")

	//deletes
	deleteItem(employeeTableName, "1")
	getItem(employeeTableName, "1")

	//updates
	createItem(employeeTableName, "2", "Jane Doe Junior", "20")
	getItem(employeeTableName, "2")

}

    We can see the above output is as expected. We had created 2 tables and we see them in the output.

    We added 2 items with empId as 1 & 2. We fetched the same items and printed the data.

    We deleted empId 1 and tried fetching it giving us an not found line.

    We updated the name for empId 2 from Jane Doe to Jane Doe Junior and after getting the item, we can see that it’s changed.

    Conclusion

    To Conclude,

    We saw how we can use Dynamo DB with Golang for commands like create table, insert, update, delete, and get.

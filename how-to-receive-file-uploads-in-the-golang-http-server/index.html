    Introduction

    Hey Folks, Today we will see how we can receive a file upload from an HTTP client to our HTTP server. We will use the curl utility to upload the file.

    We will be uploading files in 2 ways, in body, and as a form file.

    HTTP Server

    Let’s look at the server which we are creating. It will be a basic HTTP server only.

    func main() {

	server := http.Server{
		Handler: http.HandlerFunc(handle),
		Addr:    ":9000",
	}

	log.Println("starting server")
	log.Println(server.ListenAndServe())

}

func handle(w http.ResponseWriter, r *http.Request) {

	log.Println("path", r.URL.Path)

	switch r.URL.Path {
	case "/receive-file":
		upload(w, r)
	default:
		w.WriteHeader(http.StatusNotFound)
		w.Write(NotFound)
	}

}

    File in Form

    If we are uploading the file using a form, we will do it differently than getting the file in the body. We will use the FormFile method available in the request which gives us the multipart file handle.

    Let’s look at the code, We will do below 3 things,

    1. Get the file from FormFile
    2. Create a new file
    3. Copy the contents.
    func handleFileInForm(w http.ResponseWriter, r *http.Request) {
	f, fh, err := r.FormFile("file")
	if err != nil {
		log.Println("formfile error", err)
		somethingWentWrong(w)
		return
	}

	if fh.Size <= 0 {
		w.WriteHeader(400)
		w.Write([]byte("Got File length <= 0"))
		return
	}

	outFile, err := getFile(fh.Filename)
	if err != nil {
		log.Println("getFile error", err)
		somethingWentWrong(w)
		return
	}

	written, err := io.Copy(outFile, f)
	if err != nil {
		log.Println("copy error", err)
		somethingWentWrong(w)
		return
	}

	success(w)
	log.Println("Written", written)
}

    File in Body

    Here the file data will come in the body itself like a normal HTTP request so our handling of the code will be normal and we will not use any multipart here.

    func handleFileInBody(w http.ResponseWriter, r *http.Request) {

	//Check if body if empty or not
	if r.ContentLength <= 0 {
		w.WriteHeader(400)
		w.Write([]byte("Got content length <= 0"))
		return
	}

	f, err := getFile("")
	if err != nil {
		somethingWentWrong(w)
		return
	}
	defer f.Close()

	written, err := io.Copy(f, r.Body)
	if err != nil {
		log.Println("copy error", err)
		somethingWentWrong(w)
		return
	}

	success(w)

	log.Println("Written", written)
}

    Our HTTP Handler

    Let’s look at the handler. We have a simple if to check for the multipart header and we are calling the appropriate function.

    func upload(w http.ResponseWriter, r *http.Request) {

	//If its not multipart, We will expect file data in body.
	if !strings.Contains(r.Header.Get("Content-Type"), "multipart/form-data") {
		handleFileInBody(w, r)
		return
	}

	handleFileInForm(w, r)

}

    Output

    Our code is ready, Let’s try running it and see the output. We will run the below curl commands,

    1. curl localhost:9000/receive-file -d @test.txt -> Send in Request Body.
    2. curl localhost:9000/receive-file -F file=@test.txt -> Send in Form.
    3. curl localhost:9000/receive-file -F file=@empty.txt -> Send in Form.
    4. curl localhost:9000/receive-file -d @empty.txt -> Send in Request Body.

    Conclusion

    To Conclude,

    In this post, We saw how we can receive the file upload by HTTP clients in our HTTP server.

    We also saw how we can use curl to upload files to an HTTP server.
